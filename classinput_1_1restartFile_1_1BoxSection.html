processes the box section of the rst file

the box section can have 4 or 7 elements. If it has 4 elements, the box is assumed to be orthogonal. If it has 7 elements, the box is assumed to be triclinic. The second to fourth elements are the box dimensions, the next 3 elements are the box angles.

Parameters
lineElementsall elements of the line
engineobject containing the engine
Exceptions
customException::RstFileExceptionif the number of elements in the line is not 4 or 7
customException::RstFileExceptionif the box dimensions are not positive
customException::RstFileExceptionif the box angles are not positive or larger than 90°
