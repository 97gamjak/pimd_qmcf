◆ read()

void InputFileReader::read ( )

read input file

Reads input file line by line. One line can consist of multiple commands separated by semicolons. For each command the process() function is called.

Note
Also single command lines have to be terminated with a semicolon. '#' is used for comments as in all other file formats.
Exceptions
InputFileExceptionif file not found
