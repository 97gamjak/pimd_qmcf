void ThermostatSetup::setup ( )

setup thermostat

checks if a thermostat was set in the input file, If a thermostat was selected than the user has to provide a target temperature for the thermostat.

Note
the base class Thermostat does not apply any temperature coupling to the system and therefore it represents the none thermostat.
Exceptions
InputFileExceptionif no temperature was set for the thermostat
