void ParameterFileReader::read ( )

reads parameter file

Reads parameter file and according to the first word of the line, the corresponding section is called and then the line is processed. After processing the line, the section is deleted from _parameterFileSections.

Exceptions
customException::InputFileExceptionif file was not provided
customException::InputFileExceptionif file does not exist
