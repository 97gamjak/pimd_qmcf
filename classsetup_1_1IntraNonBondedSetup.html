void IntraNonBondedSetup::setup ( )

Setup intra non bonded interactions.

Setup the coulombPotential and nonCoulombPotential in the IntraNonBonded class. Then the IntraNonBonded maps vector is filled with all single intraNonBonded maps. A single intraNonBonded map is a class containing the molecule pointer and the IntraNonBonded container which represents the molecule type of the molecule pointer.
