void PotentialSetup::setupNonCoulombicPairs ( )

sets up nonCoulombic pairs in the ForceFieldNonCoulomb potential

Following steps are performed: 1) calculate energy and force cut off for each nonCoulombic pair 2) determine internal global vdw types 3) check if all self interacting non coulombics are set 4) sort self interacting non coulombics 5) check if all self interacting non coulombics are set 6) fill diagonal elements of nonCoulombicPairsMatrix 7) fill non diagonal elements of nonCoulombicPairsMatrix

Exceptions
customException::ParameterFileExceptionif not all self interacting non coulombics are set
