◆ processMolecule()

void IntraNonBondedReader::processMolecule ( const size_t  moleculeType)

processes the intra nonBonded interactions for a given molecule type

the atomIndices vector is a vector of vectors. The first index is the reference atom index. The second index is the atom index that interacts with the reference atom. The sign of the atom index indicates the type of interaction. If the sign is negative then the interaction is a 1-4 interaction and has to be scaled accordingly.

Each line should have the following format: <reference atom index> <atom index 1> <atom index 2> ... (negative atom index means 1-4 interaction)

The molecule section should end with "END" (case insensitive)

◆ read()

void IntraNonBondedReader::read ( )

reads the intra non bonded interactions from the intraNonBonded file

The function reads the intra non bonded interactions from the intraNonBonded file. It calls the processMolecule function if a molecule type is found. The molecule type can be given either via the string name or the size_t molecule type.

Exceptions
customException::IntraNonBondedExceptionif the intraNonBonded file is not provided by the user
customException::IntraNonBondedExceptionif the intraNonBonded file does not exist
customException::IntraNonBondedExceptionif the molecule type is not found
