void ManostatSetup::setup ( )

setup manostat

checks if a manostat was set in the input file, If a manostat was selected than the user has to provide a target pressure for the manostat.

Note
the base class manostat does not apply any pressure coupling to the system and therefore it represents the none manostat.
Exceptions
InputFileExceptionif no pressure was set for the manostat
