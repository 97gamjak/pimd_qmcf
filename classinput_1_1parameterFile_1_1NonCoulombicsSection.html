processes the buckingham nonCoulombics section of the parameter file and adds the buckingham pair to the nonCoulombic potential

The line is expected to have the following format:

  1. global van der Waals type 1
  2. global van der Waals type 2
  3. a
  4. dRho
  5. c6
  6. cutOff (optional); if not given or -1, the global Coulomb radius cutOff is used
