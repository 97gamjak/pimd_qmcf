.. _outputFiles:

############
Output Files
############


*******
Box File 
*******

File Type: ``.box``

Contains the three lattice parameters (*a*, *b*, *c*) and the three angles (*α*, *β*, *γ*) of the simulation box for every simulatio step in the following format:
    
    step_number *a* *b* *c* *α* *β* *γ*
